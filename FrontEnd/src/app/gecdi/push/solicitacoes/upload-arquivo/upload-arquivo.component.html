Upload de arquivo para solicitação de PUSH {{idSolicitacao_PUSH}}

Aqui você irá carregar os CPF dos clientes para a solicitação de envio de PUSH

Instruções:

  • Prepare o arquivo conforme o layout e o compacte o conteúdo no formato ZIP
  • Com o mouse clique e arraste para o retangulo pontilhado o arquivo ou use o botão [Selecione o arquivo de CPF/Variáveis para carregar] e navegue até o arquivo e clique em OK
  • Após aparecer o nome do arquivo acima do botão [Upload], clique no botão [Upload] para carregar o arquivo no sistema
  • Aguarde o carregamento e o feedback do sistema de que o 'Upload realizado com sucesso !'
  • Caso ocorram erros, de forma persistente, envie cópias de tela para o CPE GECDI02

Layouts permitidos:

  • Só CPF:
    • Arquivo ZIP contendo um único arquivo texto com enconding UTF-8
    • 1 CPF válido por linha, somente números e com as 11 posições numéricas
    • Sem espaços nem pontuações
    • Sem cabeçalho
  • CPF + Campos para mensagens dinâmicas/variáveis:
    • Arquivo ZIP contendo um único arquivo texto com enconding UTF-8
    • 1 CPF válido por linha, somente números e com as 11 posições numéricas, mais 5 campos, separados por ; (ponto e virgula) ou | (pipe) ou \t (caracter tab) ou , (virgula)
    • Sem espaços nem pontuações nos campos (CPF ou campos)
    • Sem cabeçalho
